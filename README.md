# uxr-skills

A Claude Code plugin with skills for UX researchers.

## Skills

### summarize-research-session

Produces a structured, scannable summary from user research session notes and/or transcripts. Works with interviews, usability tests, and concept tests.

**Output includes:** participant background, key quotes, themes, usability issues, hypothesis evaluation, action items, and additional observations.

## Installation

```
claude plugin add --from github:hm780/uxr-skills
```
